# examples/

Example scripts for scenario runs and CARLA experiments.

Main script:

- `run_followstop.py`: followstop scenario run harness (legacy-compatible entry).

Subdirectory:

- `carla_examples/`: standalone CARLA sample scripts and experiments.

Recommendation:

- For regular runs, prefer `python -m carla_testbed run --config ...`.
- Use `examples/run_followstop.py` when debugging low-level scenario options quickly.
