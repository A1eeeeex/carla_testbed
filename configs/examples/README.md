# configs/examples/

Small canonical examples for docs, smoke tests, and future typed config work.

These files should stay compact and generic. They should not contain Apollo-specific bridge tuning, machine-local paths, or one-off experiment overrides.

Current examples:

- `smoke.yaml`: minimal CARLA smoke config shape.

Older runnable profiles remain in `configs/io/examples/` for now and are documented as transitional compatibility profiles.
